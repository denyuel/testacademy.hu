# TestAcademy | Senior QA Mentoring & AI Automation

[![QA Pipeline](https://github.com/denyuel/testacademy.hu/actions/workflows/qa-tests.yml/badge.svg)](https://github.com/denyuel/testacademy.hu/actions)

Welcome to the **TestAcademy** Open Source Repository. This project serves a dual purpose:
1. The primary frontend landing page and enrollment mechanism for the Academy (`index.html`).
2. A complete **Enterprise Java Test Automation Reference Architecture** located in `/qa-automation/`.

## 🏗 Enterprise Automation Portfolio
If you are reviewing this repository to evaluate test architecture capabilities, please navigate to the [qa-automation module](./qa-automation) immediately.

The `qa-automation` repository demonstrates mastery over:
*   **Java 17 & JUnit 5 Matrix Capabilities**
*   **API Automation:** REST Assured, JSON Schema Validation, Jackson POJO Data-Driven testing (BVA).
*   **UI Automation:** Selenide / Selenium, Page Object Model (POM), AssertJ Soft Assertions.
*   **Database Interaction:** JDBC connections validating against H2 In-Memory DBs.
*   **CI/CD DevSecOps:** Integrated GitHub Actions executing NodeJS mock middlewares natively.

## 🚀 Deployment
*   **Frontend:** The landing page is stateless and can be launched directly via `localhost` or served statically.
*   **Pipeline:** CI validations trigger strictly on Pull Requests against the Main branch.
